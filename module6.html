Quantitative Data Analysis III

Correlation, regression, and chi-square analysis with SPSS

ℹ️  Overview

Welcome to this module! In this learning module, you will learn how to analyze quantitative data, focusing on correlation and regression analysis with SPSS. In addition to these two analyses, you will learn how to analyze categorical data with SPSS.

📈 Correlation Analysis

Used to quantify the association between two continuous variables — e.g., between an independent and dependent variable.

📉 Regression Analysis

Assesses the relationship between an outcome variable and one or more predictors. Dependent variable = "y"; independent = "x".

🔢 Chi-Square Test

Used for testing relationships between categorical variables. Null hypothesis: no relationship exists between variables.
