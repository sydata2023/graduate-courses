Fundamentals of statistics and setting up data in SPSS

ℹ️  Overview

Welcome to this module! In this learning module, you will learn the fundamentals of statistics by watching videos and online reading. In addition, you will practice how to set up raw data in SPSS (Statistical Package for the Social Sciences). SPSS is a very useful tool to analyze numeric data for all researchers, not just for quantitative researchers.

💾
Action required: Download and install SPSS before starting the learning activities. You will need it to complete all 4 activities in this module.
